Proyecto de E-commerce Talento Tech

El proyecto cuenta con las siguientes estructuras:

INDEX compuesto por
  + index html: Es la estructura de la pagina inicial 
  + index js: controla la funcionalidad e interactividad de la pagina, como por ejemplo las acciones del menú responsive.
    la interactividad de los botones y las distintas alertas.
  + styles css : es la planilla de estilos general, aplicada a la mayoria de las secciones de la pagina web.

PRODUCTOS
  +productos html estructura de la pagina donde mostramos los prodoctos ofrecidos en la tienda, en formato de lista
  y con adaptacion para disntitos dispositivos mobiles.
  +productos js se encarga de la funcionalidad de la seccion, como por ejemplo el uso de la api FAKESTORE.API para obtener
  el listado de productos que se muestran. 
  +productos css

AGREGAR PRODUCTOS
+agregarProductos html estructura general que contiene el formulario de agregado de productos nuevos, junto con la posibilidad de arrastrar y soltar imágenes
no tenemos base de datos por lo tanto los datos por el momento no son almacenados pero es una funcionalidad útil para mantener la pagina actualizada.
+agregarProductos js encontramos las funciones relacionadas con la carga de datos en el fornmulario de agregar productos. 
+agregarProductos css en esta pagina tenemos los ajustes pertinentes a la pagina asi tambien como el estilo aplicado al formulario. 

SESSION Y REGISTER 
+session html y register html son las estructuras de ambas paginas que contienen los formularios de inicio de sesion y de registro respectivamente
+StylesSession css y StylesRegister css contienen los estilos particulares de ambas paginas ya que el formato de las mismas esta adaptado al contenido 
para hacerlas visualmente mas atractivas y elegantes. 

SUCURSALES 
sucursales html y sucursales css en estos documentos se encuentra la estructura definida de la sección sucursales, donde tenemos el espacio para mostrar 
nuestra ubicación para poder brindarle al publico los detalles sobre donde encontrarnos. 
Aquí por medio de un iframe insertamoas un mapa para poder posteriormente ingresar la ubicación física. 

CONTACTANOS 
contactanos html y contactanos css son los archivos que dan forma a la seccion de contactanos, en donde los clientes tienen la posibilidad de ingresar 
sus datos y dejarnos un mensaje o consulta para que luego pueda ser respondida por nuestro equipo via e-mail. 

COMPRA 
compra js este archivo es fundamental, ya que regula el comportamiento del modal compra, que es el que se encarga de la captura de datos y simulacion de 
comrpa de los productos por individual. Este archivo se encuentra en todos los documentos que posean los botones interactivos de comprar ahora, en este 
momento ubicados en la seccion productos e inicio. 

CARRITO
carrito js es el archivo encargado de la regulacion del comportaminto del modal de carrito, en donde vemos los productos que agregamos a nuestra compra, 
cuando queremos comprar mas de 1 articulo, en este modal en forma de popup tenemos para ingresar los datos, e mail, medio de pago, vemos el total y 
tambien podemos controlar la cantidad de productos y cuales eliminar o aumentar la cantidad de los mismos. al final del mismo también vemos el total 
en efectivo y también es el que se encarga de realizar las comprobaciones correspondientes para evitar que se realicen compras sin controlar los datos.
